The Debian Foreman staging repo

This repo is for building and testing scratch packages before pushing them to the real repos

Every user will have his packages pushd to his own component, based on the github repository used.
So if you specified 'ares' as the github repo, and 'wheezy' as the dsitro, you package is at: deb http://debstaging.theforeman.org/ wheezy ares Enjoy
